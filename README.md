## 详细行程

D1 : 西安到东京 ，看夜景 ， 日本铁塔？
D2 : 东京市内-- 浅草寺，涩谷，自由之丘，
D3 : 镰仓 ，当日返回东京
D4 : 河口湖，住一晚
D5 : 河口湖 ，放回东京
D6 : 东京回西安
